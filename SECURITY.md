# Security Policy

## Reporting a Vulnerability

Do NOT open a public GitHub issue for security vulnerabilities.

Email: oxilic05701@gmail.com with subject: `[SECURITY] GitPhone`

We will respond within 48 hours. Once confirmed, we will issue a fix
and credit you in the release notes.

## Known MVP Limitations
- GitHub tokens are stored as plain text (encryption is roadmap item)
- No rate limiting on API routes yet
See `05_SECURITY.md` in `.context/` for full details.
